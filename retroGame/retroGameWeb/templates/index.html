What's Next Game Database

What We Do:

What’s Next Game Database is a collection of data designed around the idea of finding new games to play. What’s Next allows fans of video games to peruse and select a number of different factors in order to find games they might like to play. This application is free of a lot of the unnecessary bog that tends to weigh down similar databases. There is no arbitrary numeric rating system, no need to create accounts, it’s just about the games and finding more to play.

How To Use:

This application allows users the option to search for new games to play. To search for a new game, go to the search page and select values you would like to return results for. To have a game recommended to you, click the recommend tab and follow the instructions.
